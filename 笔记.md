#### Visio studio 卸载
1. 卸载 Visual Studio 本身
通过控制面板卸载：

打开控制面板。
选择 "程序和功能"。
找到 Visual Studio 版本并选择卸载。
使用 Visual Studio Installer 卸载：

打开 Visual Studio Installer（通常在开始菜单中可以找到）。
找到已安装的 Visual Studio 版本，点击 "更多" 按钮，然后选择 "卸载"。
2. 删除 Visual Studio 安装文件夹
默认情况下，Visual Studio 会安装在以下位置（根据你的安装路径可能会有所不同）：

C:\Program Files (x86)\Microsoft Visual Studio
删除此文件夹中的所有内容。需要管理员权限。

3. 删除 MSBuild 文件夹
删除与 MSBuild 相关的文件夹，通常位于：

C:\Program Files (x86)\MSBuild
4. 删除 Visual Studio Installer
如果你不再需要 Visual Studio Installer，可以将其卸载：

打开控制面板，进入 "程序和功能"。
找到 "Visual Studio Installer" 并卸载。
5. 删除 NuGet 缓存和全局包
删除与 NuGet 相关的文件夹，通常位于：

%userprofile%\.nuget
%userprofile%\.nuget\packages
6. 删除 Visual Studio 项目模板和项目缓存
删除与 Visual Studio 项目模板和项目缓存相关的文件夹，通常位于：

%userprofile%\Documents\Visual Studio 20xx
7. 删除注册表项
删除与 Visual Studio 相关的注册表项。请注意，修改注册表有风险，建议先备份注册表。

打开注册表编辑器（regedit）：

按 Win + R 打开运行对话框，输入 regedit 并按回车。
查找并删除以下注册表项：

HKEY_CURRENT_USER\Software\Microsoft\VisualStudio
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\VisualStudio
HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Microsoft\VisualStudio
还要删除一个VSCmomon
8. 删除环境变量
检查系统环境变量中是否有 Visual Studio 相关的变量，并手动删除：

右键点击 "此电脑" 或 "我的电脑"，选择 "属性"。
点击 "高级系统设置"。
在系统属性窗口中点击 "环境变量"。
查找并删除与 Visual Studio 相关的变量。
9. 删除系统服务和计划任务
检查并删除与 Visual Studio 相关的系统服务和计划任务：

打开任务计划程序：按 Win + R 打开运行对话框，输入 taskschd.msc 并按回车。
检查并删除与 Visual Studio 相关的任务。
10. 删除用户配置文件和设置
删除用户配置文件和设置，通常位于：

%appdata%\Microsoft\VisualStudio
%localappdata%\Microsoft\VisualStudio
%localappdata%\Microsoft\VSCommon
还要删除一个VSCapplication
通过上述步骤，你可以手动彻底删除 Visual Studio 及其相关组件。完成这些步骤后，建议重启计算机以确保所有更改生效。

注意事项
备份数据：在执行任何删除操作之前，确保备份你的项目和相关数据。
管理员权限：执行这些步骤时，可能需要管理员权限。
风险提示：手动删除注册表项和系统文件具有一定风险，错误操作可能影响系统稳定性。确保谨慎操作。
